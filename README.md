Pasos pasos para la instalación de Laravel .

Requisitos :
 -  XAMPP: Utima versión  ( Se debe clonar  el arhivo en la htdocs)
 -  Node.js
 -  Composer 

Comando necesarios
- npm install
- npm run dev
- composer install
- php artisan migrate
- php artisan serve
- php artisan key:generate
